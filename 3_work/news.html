VTB API hackathon 2025: Создай инновации с открытыми API

Опубликовано: 27 октября 2025 Автор: Редакция IT Eventus

27 октября – 22 ноября 2025 года пройдет VTB API hackathon 2025 — специализированный хакатон по открытым API от ВТБ. Призовой фонд составляет 2.500.000 рублей! Ожидается участие сотен разработчиков из России и стран СНГ.

Что вас ждет

VTB API hackathon 2025 — это уникальная возможность создать инновационные решения с использованием открытых API ВТБ. Формат мероприятия: онлайн и офлайн. Вы сможете работать в команде, использовать передовые банковские API и представить свой проект жюри из ведущих экспертов индустрии.

Треки хакатона:

  • Мультибанк — создание решений для работы с несколькими банками через единый интерфейс
  • Защита API — разработка систем безопасности и защиты API от различных угроз
  • Оркестр из API — создание комплексных решений, объединяющих различные API в единую экосистему

Призовой фонд

Общий призовой фонд составляет 2.500.000 рублей! Победители получат денежные призы, возможность стажировки в ВТБ, менторскую поддержку и помощь в развитии проекта. Дополнительные призы от партнеров мероприятия.

Особенность хакатона: Все участники получат полный доступ к API ВТБ, документацию, облачные ресурсы для разработки и менторскую поддержку от экспертов банка.

Формат мероприятия

Хакатон проходит в несколько этапов:

  1. Регистрация и отбор — подача заявок и формирование команд
  2. Онлайн-этап — разработка проектов в дистанционном формате
  3. Офлайн-финал — презентация лучших проектов перед жюри
  4. Менторские сессии — консультации с экспертами ВТБ
  5. Награждение победителей — церемония закрытия

Регистрация

Регистрация на хакатон уже открыта и абсолютно бесплатна! Количество мест ограничено, поэтому не упустите возможность. Регистрация доступна как для команд, так и для индивидуальных участников, которые смогут найти команду на мероприятии.

Не упустите возможность!

Присоединяйтесь к VTB API hackathon 2025 и создайте проект с призовым фондом 2.500.000 рублей! Работайте с банковскими API, получайте менторскую поддержку от экспертов ВТБ и выигрывайте призы!
